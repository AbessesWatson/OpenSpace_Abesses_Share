Bonjour, ceci est le partage de mon travail pour le projet Open Space RP 

Il faut savoir ce projet a été le fruit du travail de plusieurs personnes. Les ressources ci présente ne peuvent donc que très difficilement être installé seul. Elles ont des dépendances.



DEPENDANCES :

La dépendances principal est le framework QBX. Si vous voulez travailler sans il y aura alors un très grand nombre de modification à faire dans quasiment tout les scripts. 

La seconde grande dépendance est à la map d'Open Space faite majoritairement par Eazy_V. Je vous recommande de le contacter si vous voulez travailler dessus. Il faut savoir que dans la config d'un grand nombre de script, des props custom de cette map sont appliqué. A vous des modifier les configurations si vous voulez utiliser des scripts sans la map. 

Le travail de Proky a créé un dependances avec l'UI d'Open Space. A vous de le contacter ou bien de contourné les erreurs avec de petite modifications. (A noté que le mute lorsque on est dans le coma fait partie de ses dependances)

Le script de VakzOs vkz-cin est utiliser dans le script de smallstuff. 

Pour ce qui est des standalones. Les scripts où il y a une dependance sont: 
- illenium-appearance (très légère et contournable)
- scully_emotemenu (principalement pour le script smallstuff
